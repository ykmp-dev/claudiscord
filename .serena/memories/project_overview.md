# claudiscord Project Overview

## プロジェクト目的
Discord経由でローカル/VPS開発環境のLLM（Claude Code CLI / GLM-5）を操作できるボット。

## 主な機能
- DiscordスラッシュコマンドでのLLM操作
- Claude Code CLI と GLM-5 (z.ai) の切り替え
- ツール実行の承認フロー
- チャンネルごとのセッション管理

## 技術スタック
- Node.js + TypeScript
- Discord.js
- @anthropic-ai/claude-agent-sdk
- z.ai API (GLM-5)

## デプロイ先
- VPS
